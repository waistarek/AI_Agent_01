# AI Agent (LangChain)

Small research agent using Gemini with tool-calling (DuckDuckGo, Wikipedia, file save).

## Quickstart
1) Python 3.11+ and a virtualenv
2) `pip install -r requirements.txt`
3) Create `.env` with `GOOGLE_API_KEY=...`
4) `python main.py`

## Tools
- DuckDuckGo search
- Wikipedia summaries
- Save research output to file
